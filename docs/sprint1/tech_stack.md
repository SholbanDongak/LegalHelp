
# Технологический стек

## Backend
- Язык: Python 3.11
- Фреймворк: FastAPI
- База данных: PostgreSQL
- ORM: SQLAlchemy
- Миграции: Alembic

## Frontend (опционально)
- Вариант 1: React + TypeScript
- Вариант 2: Telegram Bot (более простой)

## DevOps
- Контейнеризация: Docker
- CI/CD: GitHub Actions
- Регистр: Docker Hub

## Интеграции
- Google Calendar API
- Яндекс.Календарь

## Хранение данных
- Задачи: PostgreSQL
- Таймеры: in-memory / Redis
