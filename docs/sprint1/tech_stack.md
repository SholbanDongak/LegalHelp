# Технологический стек LegalHelp (РФ-совместимый)

## Backend
| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| Язык | Python 3.11 | Простота, богатая экосистема |
| Фреймворк | FastAPI | Высокая производительность, async |
| База данных | PostgreSQL | Надёжность, ACID, бесплатно |
| ORM | SQLAlchemy | Стандарт в Python |
| Миграции | Alembic | Управление схемой БД |

## OCR (распознавание текста)
| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| Основной | EasyOCR | Лучший для русских документов |
| Резервный | Tesseract | Бесплатно, работает офлайн |
| PDF | pdfplumber | Извлечение текста из PDF |

## AI / LLM
| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| Основной | YandexGPT 5.1 Pro | Работает в РФ, качественный русский язык |
| Цена | 0,40 ₽ / 1K токенов | Дешевле OpenAI в 10 раз |
| Соответствие | 152-ФЗ | Персональные данные не покидают РФ |

## DevOps
| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| Контейнеризация | Docker | Воспроизводимость, изоляция |
| CI/CD | GitHub Actions | Автоматическое тестирование |
| Хостинг | Yandex Cloud / Timeweb | РФ-провайдеры |
