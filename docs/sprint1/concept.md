
# Концепция продукта

## Название
Трекер задач с таймерами (Pomodoro Tracker)

## Основная идея
Приложение для управления задачами с встроенными таймерами Pomodoro.

## Основной функционал
1. Создание задач с дедлайнами
2. Таймеры для контроля времени
3. Уведомления о дедлайнах
4. Графики продуктивности
5. Интеграция с календарём

## Технологический стек (предварительно)
- Backend: Python + FastAPI
- Frontend: React / Telegram Bot
- База данных: PostgreSQL
- Контейнеризация: Docker
- CI/CD: GitHub Actions
