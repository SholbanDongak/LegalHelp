# Концепция продукта LegalHelp

## Название
LegalHelp — AI-ассистент для ответов на входящие документы

## Основная идея
Автоматическая генерация юридически корректных черновиков ответов на:
- Запросы госорганов (ФНС, прокуратура, трудовая инспекция, Роскомнадзор, ФАС)
- Судебные акты (определения, иски, запросы)
- Письма контрагентов (претензии, оферты, рекламации)
- Обращения граждан и жалобы

## Основной функционал MVP
1. Загрузка скана/PDF входящего документа
2. OCR-распознавание текста (EasyOCR + Tesseract)
3. Классификация типа документа (8 категорий)
4. AI-генерация черновика ответа (YandexGPT)
5. Скачивание ответа в Word/PDF
6. Ручной ввод текста (запасной вариант)

## Технологический стек (РФ-совместимый)
| Компонент | Технология |
|-----------|------------|
| Backend | Python 3.11 + FastAPI |
| База данных | PostgreSQL |
| OCR | EasyOCR + Tesseract |
| LLM | YandexGPT (API Яндекс.Облака) |
| Контейнеризация | Docker |
| CI/CD | GitHub Actions |

## Целевая аудитория (MVP)
- Малый бизнес (ООО, ИП без штатного юриста)
- Юридические фирмы (автоматизация рутины)
- Отделы закупок (ответы на жалобы в ФАС)
- Руководители (переписка с контрагентами)

## Уникальное торговое предложение (УТП)
- Генерация ответа за 1 минуту (против 3-4 часов ручной работы)
- Стоимость 0,5-1 ₽ за ответ (через YandexGPT)
- Работает в РФ без VPN, соответствует 152-ФЗ
