
# LegalHelp — Юридический AI-ассистент для подготовки ответов и процессуальных документов

**Автор:** Шолбан Донгак  
**Репозиторий:** [GitHub](https://github.com/SholbanDongak/LegalHelp)

## Описание
LegalHelp – платформа на основе RAG и локальной LLM (Qwen2.5). Помогает:
- Отвечать на запросы госорганов (ФНС, прокуратура и др.).
- Составлять ответы контрагентам.
- Готовить процессуальные документы: возражения на судебные приказы, апелляционные/кассационные жалобы, жалобы по КоАП.

**Локальность → конфиденциальность (152-ФЗ).**

## Технологии
- Бэкенд: FastAPI + ChromaDB (304 тыс. законов)
- LLM: Ollama + Qwen2.5
- Гибридный поиск: BM25 + эмбеддинги
- Фронтенд: React + Vite
- OCR: EasyOCR

## ML-метрики (на 25 запросах)
| Стратегия | Precision@5 | MRR |
|-----------|-------------|-----|
| Векторный поиск | 0.62 | 0.58 |
| Гибридный поиск | 0.87 | 0.84 |

| Конфигурация | Верность ответов (faithfulness) |
|--------------|--------------------------------|
| Без RAG | 66% (34% галлюцинаций) |
| RAG (векторный) | 89% |
| RAG + гибридный | 92% |

**Вывод:** гибридный RAG снижает галлюцинации, критично для юриспруденции.

## Запуск
```bash
ollama pull qwen2.5:7b-instruct
ollama serve
cd ~/LegalHelp && source legal_env/bin/activate && pip install -r requirements.txt && python -m uvicorn src.main:app --reload --port 8000
cd ~/legal-frontend && npm install && npm run dev
